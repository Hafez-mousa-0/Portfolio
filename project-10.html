Frontline Intelligence Service

Project Overview

I worked on creating the FIS landing and onboarding experience to simplify first-time engagement, guide users through account setup, and deliver a clear, intuitive entry point into the FIS experience. The goal was to increase first-time conversion and engagement while reducing onboarding friction by combining file upload, prompt entry, and account creation into one unified flow.

Landing Page Experience

The redesigned landing page features a clean, focused interface that immediately communicates value:

  • Headline: "What will you discover today?" with subtext "Uncover the hidden intelligence in your data and turn it into action."
  • Central Chat Interface: A prominent chat box with "+" icon for file upload, text input for prompts, and "Explore" button
  • Dynamic Typing Animation: The chat box displays rotating example prompts that demonstrate FIS capabilities
  • Feature Showcase: Three visual boxes highlighting Connect, Analyze, and Act capabilities with the tagline "From raw data to real decisions â€” instantly."

User Input Scenarios & Flow

I designed multiple user journey paths to handle different entry points:

  • Prompt Only: Users entering prompts without files are guided to select a file or choose from sample datasets
  • File Upload Only: Users uploading files are prompted to sign up, then receive general analysis of their dataset
  • Both Provided: Seamless flow when both file and prompt are provided
  • File First: Clicking "+" opens file selector, then prompts account creation

Account Creation & Onboarding

Streamlined sign-up process with validation and error handling for username, password, and password confirmation. The flow ensures users can proceed only after successful registration.

Post-Analysis Experience

After analysis, users are guided through:

  • Data Table Review: Clean, scrollable preview with auto-detected columns, sorting, filtering, and sampling
  • Prompt Review: Display and edit functionality for user-entered or system-generated prompts
  • Workspace Transition: Seamless transition to the main Workspace screen after confirmation
